# Hoe gaan we de app uiteindelijk gebruiken?

## Niet starten met App Store

Voor intern gebruik starten we niet met een App Store-app.

We starten met een webapp/PWA:

- werkt op iPhone
- werkt op iPad
- werkt op laptop
- werkt op computer
- openen via link
- toevoegen als icoon op beginscherm
- sneller te bouwen
- goedkoper
- makkelijker aan te passen

## Praktisch gebruik

Tijdens klantcontact:

1. klant aanmaken
2. project aanmaken
3. afspraak/inmeting plannen
4. productregels toevoegen per ruimte
5. maten invullen in mm
6. foto's toevoegen
7. uitvoering kiezen
8. prijs controleren
9. offerte maken
10. PDF sturen

Tijdens montage:

1. project openen
2. productregels bekijken
3. foto's bekijken
4. aandachtspunten lezen
5. montage uitvoeren
6. status aanpassen
